PASS 처음 시작하기
====================

웹 개발이나 Git을 처음 사용하는 분은 아래 문서를 먼저 읽어 주세요.

상세 가이드:
docs/BEGINNER_GUIDE_KO.md

GitHub에서도 위 파일을 클릭하면 별도 프로그램 없이 읽을 수 있습니다.

가장 간단한 실행 순서
---------------------

[Windows]
1. VS Code에서 PASS 폴더를 엽니다.
2. 상단 메뉴에서 Terminal > New Terminal을 누릅니다.
3. 터미널에 아래 명령을 입력합니다.

   .\scripts\setup_windows.bat

4. 설치가 끝나면 아래 명령을 입력합니다.

   .\scripts\start_windows.bat

5. 브라우저에서 http://localhost:5173 을 엽니다.

[Mac]
1. VS Code에서 PASS 폴더를 엽니다.
2. 상단 메뉴에서 Terminal > New Terminal을 누릅니다.
3. 최초 한 번 아래 명령을 차례대로 입력합니다.

   chmod +x scripts/*.command
   ./scripts/setup_mac.command

4. 설치가 끝나면 아래 명령을 입력합니다.

   ./scripts/start_mac.command

5. 브라우저에서 http://localhost:5173 을 엽니다.

주의
----
- setup은 컴퓨터마다 최초 한 번만 실행합니다.
- start는 PASS를 사용할 때마다 실행합니다.
- 실행 중인 터미널 또는 명령 프롬프트 창을 닫으면 PASS도 종료됩니다.
- 처음 설치할 때는 Python 패키지와 Frontend 패키지를 내려받기 때문에
  인터넷 또는 사내 패키지 저장소 연결이 필요합니다.
- 자세한 설치, GitHub 연결, 오류 해결, 폴더/파일 설명은
  docs/BEGINNER_GUIDE_KO.md에 있습니다.
